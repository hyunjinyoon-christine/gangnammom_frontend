# 강남엄마 사전 과제 
## Vuex proejct
- 해당 위치(./gangnammom)에서 실행해주세요
```
# 최초 설치 후 패키지 설치
npm install

# 로컬에서 개발할때 사용
npm run serve

# 배포 전에 빌드를 한번 해줍니다
npm run build
```
## 사용된 라이브러리
- vue-router
- vuex-persistedstate
- vue-meta
- vue-infinite-scroll
- vue-cli
